# SignAI Client

This React app powers the SignAI Text/Audio to ISL conversion experience.

## Scripts

In this directory, run:

### `npm start`
Starts the app on `http://localhost:5000/`.

### `npm run dev`
Alias of start, also runs on `http://localhost:5000/`.

### `npm run build`
Builds a production bundle.

### `npm test`
Runs the test suite.

## Product Behavior
- `/` opens the conversion UI directly.
- The app uses SignAI branding and unified styling to match the main frontend product shell.
